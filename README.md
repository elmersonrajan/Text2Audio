# Text2Audio

Convert text into speech audio files using Python.

## Features

* Convert text to audio
* Generate MP3 output
* Simple and lightweight
* Easy to customize

## Installation

Clone the repository:

```bash
git clone https://github.com/elmersonrajan/Text2Audio.git
cd Text2Audio
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the script:

```bash
python text2audio.py
```

The generated audio file will be saved in the project directory.

## Requirements

* Python 3.12 or Python 3.13 recommended
* Internet connection (required for Google Text-to-Speech)
